# 1С:Предприятие 8.3 - Практическое пособие разработчика (1C EDT)

Проект информационной базы «Услуги», выполненный в 1C:EDT.
Каждая папка содержит состояние проекта после соответствующего занятия.

## Структура

| Папка | Занятие | Описание |
|-------|---------|----------|
| `Занятие_2` | Глава 2 | Подсистемы и общие картинки |
| `Занятие_3` | Глава 3 | Справочники |
| `Занятие_4` | Глава 4 | Документы |
| `Занятие_5` | Глава 5 | Регистры накопления |
| `Занятие_6` | Глава 6 | Простой отчёт |
| `Занятие_7` | Глава 7 | Знакомство со встроенным языком |
| `Занятие_8` | Глава 8 | Дополнительный сервис в формах документа |
| `Занятие_9` | Глава 9 | Периодические регистры сведений |

## Объекты конфигурации

- **Подсистемы:** Бухгалтерия, Оказание услуг, Предприятие, Расчёт зарплаты, Учёт материалов
- **Общие картинки:** Бухгалтерия, Зарплата, Материалы, Предприятие, Услуги
- **Справочники:** Клиенты, Номенклатура, Склады, Сотрудники, Товары
- **Документы:** Приходная накладная, Оказание услуги
- **Регистр накопления:** Остатки материалов
- **Регистр сведений:** Цены
- **Отчёт:** Материалы
- **Общие модули:** Серверный, Работа с документами, Работа со справочниками
